1. ¿Qué es esta plataforma?

La Plataforma Financiera de Grupo La Plaza es una aplicación web desarrollada completamente a medida para la gestión financiera, operativa y comercial del grupo. Centraliza en un solo lugar toda la información que antes estaba dispersa en múltiples planillas Excel, permitiendo trabajar en tiempo real desde cualquier dispositivo con acceso a internet.

Acceso: grupolaplaza.vercel.app
Funciona en computadoras, tablets y celulares. Solo requiere usuario y contraseña.

2. Módulos de la plataforma

📊 Dashboard — Panel de Control

Pantalla principal que muestra el estado financiero consolidado del grupo en tiempo real. Incluye:

  • KPIs financieros clave de todas las empresas
  • Resumen de efectivo, cheques en cartera y rechazados
  • Alertas de vencimientos y compromisos próximos
  • Cotización del dólar blue actualizada
  • Tasas bancarias y de inflación en vivo
  • Rendimiento de inversiones FCI

📥 EXCEL — Importación de datos

Centro de importación de planillas. La plataforma lee directamente los Excel de trabajo sin necesidad de reformatearlos:

  • Minuta Diaria — posición de efectivo, banco, erogaciones y cheques por estación
    • Cartera de chequesCheques recibidos con todos sus datos
  • Extractos bancarios — movimientos de cuentas (Galicia, etc.)
  • Gastos contables — exportación del sistema contable
  • Rechazados — listado de cheques rechazados
  • Petroleo / Flota / Tarjetas — planillas operativas

🔍 Lector de Documentos

Módulo de inteligencia artificial que analiza documentos automáticamente:

  • Acepta PDF, imágenes (JPG/PNG) y Excel
  • Extrae: categoría, empresa, período, importes, vencimientos y movimientos detallados
  • Útil para procesar facturas, estados de cuenta y contratos sin tipear datos manualmente

3. Datos y seguridad

Almacenamiento en la nube

Todos los datos se guardan en la nube usando Supabase (base de datos PostgreSQL). Esto significa:

  • Los datos están disponibles desde cualquier dispositivo
  • No se pierden si se borra el navegador o se cambia de computadora
  • Respaldo automático permanente

Sistema de usuarios y roles

RolPermisos
AdministradorAcceso completo a todos los módulos, gestión de usuarios, importación de datos
UsuarioAcceso a los módulos asignados según su área (Finanzas, Operaciones, Administración, etc.)

El acceso se controla con usuario y contraseña. Cada persona del equipo tiene sus propias credenciales.

5. Flujo de trabajo diario

Mañana — Apertura del día

  1. Ingresar a grupolaplaza.vercel.app con usuario y contraseña
  2. El Dashboard muestra el estado del día anterior y las alertas activas
  3. Revisar Recordatorios para vencimientos del día o semana
  4. Consultar posición de Cheques para ver cobros esperados

Durante el día — Carga de datos

  1. El tesorero importa la Minuta Diaria desde Excel → sección EXCEL → Importar Minuta
  2. Automáticamente se actualizan: efectivo por estación, disponible banco, erogaciones, cheques cartera y rechazados
  3. Si hay cheques nuevos, se importa la Cartera de Cheques desde Excel
  4. Si llegaron extractos bancarios, se importan desde la sección Extractos

Análisis y reportes

  • Unidades de negocio — comparar efectivo, banco y erogaciones entre sucursales
  • Análisis de Gastos — ver qué cuentas contables se están moviendo más
  • Cash Flow — proyectar la posición de los próximos días
  • Pasivos — controlar vencimiento de cuotas de préstamos
